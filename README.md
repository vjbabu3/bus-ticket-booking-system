 Bus Ticket Booking System

A simple Bus Ticket Booking System developed using Java, JDBC, and MySQL. This project allows users to book bus tickets, check seat availability, cancel bookings, and manage reservation details through a console-based application.

 Features

- User Registration and Login
- View Available Buses
- Check Seat Availability
- Book Bus Tickets
- Cancel Bookings
- View Booking Details
- MySQL Database Connectivity using JDBC
- Console-Based Interface

 Tech Stack

- Java
- JDBC
- MySQL
- SQL
- Object-Oriented Programming (OOP)

 Installation
 Clone the Repository

```bash
git clone https://github.com/vjbabu3/bus-ticket-booking-system.git
```
 Open the Project

Open the project in any Java IDE such as:

- Eclipse
- IntelliJ IDEA
- VS Code
 Configure MySQL

Create a database:

```sql
CREATE DATABASE bus_booking;
```

Update the database username and password in the database connection file.

Run the project.

 How It Works

1. Register or Login
2. View Available Buses
3. Select a Bus
4. Check Seat Availability
5. Book Ticket
6. View Booking Details
7. Cancel Ticket (Optional)
